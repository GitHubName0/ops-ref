# Ops Ref 工具质量指标

> 基于工程控制论框架定义，用于每次变更前后的反馈测量。

## 核心指标

| 指标 | 测量方法 | 目标值 | v1.2.0 | v1.3.0 |
|------|---------|--------|--------|--------|
| 搜索准确率 | `ops search <已知存在词>` 是否返回结果 | 100% | Bug: 计数0 | ✓ 15条 |
| 搜索完整性 | 搜索结果是否覆盖所有含该词章节 | 100% | ✓ | ✓ |
| 搜索计数正确性 | 匹配计数与实际匹配数一致 | 100% | ✗ (永远0) | ✓ |
| 无匹配提示准确性 | 搜索不存在的词是否正确提示 | 正确提示 | ✓ (bug: 有时也显示) | ✓ |
| 章节完整性 | `ops check` 通过率 | 100% | 无此命令 | ✓ 7/7 |
| 交互菜单可用性 | 不破坏 scrollback | 保留 | ✗ clear清屏 | ✓ |
| 代码DRY度 | CHAPTERS定义次数 | 1 | 2 (ops+build.sh) | 1 (chapters.sh) |
| 自诊断能力 | `ops check` 可用 | ✓ | ✗ | ✓ |
| 版本查询 | `ops --version` | ✓ | ✗ | ✓ |
