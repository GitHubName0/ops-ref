# Ops Ref AI 质量指标

> 基于工程控制论框架，定义 → 测量 → 优化 → 验证

## 核心指标

| 指标 | 测量方法 | v0.1.0 | v0.2.0 | 目标 |
|------|---------|:---:|:---:|:---:|
| 答案相关性 | 10 个常见问题人工评估 | 6/10 | 待测 | ≥ 8/10 |
| 命令准确性 | 返回命令是否针对问题 | 部分答非所问 | 按问题关键词排序 | ≥ 8/10 |
| 搜索覆盖率 | 知识块检索命中率 | ~70% | 三级回退 | ≥ 90% |
| 响应速度 | 输入→回答 | 400ms(人为) | < 10ms | < 100ms |
| 回答结构 | 是否有步骤/摘要/命令/来源 | 有但不清晰 | 📌标题+摘要+🔧命令+📖来源 | ✓ |
| 用户反馈 | 反馈机制 | 无 | 👍/👎 存储到 localStorage | ✓ |
| 弱匹配过滤 | 低分答案是否展示 | 无阈值 | 最佳分<1 则提示无结果 | ✓ |
| 知识块粒度 | 每章节知识块数 | 283(粗) | 255(H3级别分割) | 200-300 |
| 文件大小 | HTML 产物 | 265KB | 257KB | < 300KB |

## 差距分析

| 差距 | 表现 | 根因 |
|------|------|------|
| 答非所问 | 问磁盘返回网络命令 | 命令提取不按问题筛选 |
| 响应慢 | 人为 400ms 延迟 | 无意义的 setTimeout |
| 无反馈 | 不知道回答好不好 | 缺反馈机制 |
| 知识块太粗 | 一个块含多个主题 | 按 H3 分块但超限时未细分 |
| 弱匹配也出结果 | 得分极低的内容也展示 | 缺分数阈值 |
